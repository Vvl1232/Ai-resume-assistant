# 🚀 AI Career Assistant (ATS + LLM Powered)

An AI-powered Career Assistant that analyzes resumes, compares them with job descriptions, and provides intelligent feedback using NLP, embeddings, and LLMs.

## 🔥 Features

- 📄 Resume Parsing (PDF)
- 🎯 ATS Match Score (Semantic Matching)
- 🧠 Skill Gap Analysis
- 🤖 AI Suggestions using LLM
- 🌐 Deployed on Hugging Face Spaces

## 🧠 Tech Stack

- Python
- Streamlit
- Hugging Face Transformers
- Sentence Transformers (Embeddings)
- Scikit-learn
- FAISS (Vector Search)

## ⚙️ How It Works

1. Extract text from resume (PDF)
2. Convert resume & job description into embeddings
3. Compute similarity score using cosine similarity
4. Identify missing skills
5. Generate AI-based suggestions using LLM
